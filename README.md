# F.U.C.K. Forum

**不要只是幻想，去专注、理解、创造并坚持。**

一个从输入到输出的完整成长闭环社区，基于 GitHub Discussions 构建的纯前端论坛。

## 🎯 论坛分区

- **[🎯 Focus - 专注区](./categories/focus/)** - 目标与精力管理
- **[💡 Understand - 洞察区](./categories/understand/)** - 知识输入与深度思考  
- **[🚀 Create - 创造区](./categories/create/)** - 实战与作品展示
- **[💪 Keep - 坚持区](./categories/keep/)** - 打卡与复盘反思

## 🚀 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建产品版本
npm run build

# 预览构建后的版本
npm run preview
```

## 📚 分区简介

### F - Focus 专注区
目标与精力管理。寻找番茄钟搭子、分享时间管理工具、立下 Flag、戒除数字瘾。拒绝分散注意力。

### U - Understand 洞察区
知识输入与深度思考。发布硬核教程、分享读书笔记、提出深度问题。只谈底层逻辑，拒绝碎片化。

### C - Create 创造区
实战与作品展示。Build in Public，分享代码/设计/文章/原型。寻求真实反馈和合伙人。

### K - Keep 坚持区
打卡与复盘反思。建立长期打卡贴、分享失败经历、互相监督。从反思中持续成长。

## 💬 参与讨论

所有讨论基于 [GitHub Discussions](https://github.com/AL-sama0/al_discuss/discussions) 进行，支持：
- ✅ Markdown 完整语法和代码高亮
- ✅ 实时评论同步
- ✅ GitHub 账号直接登录
- ✅ 开放透明的讨论历史

## 🛠 技术栈

- **VitePress** - 静态网站生成器
- **Vue 3** - 前端框架
- **Giscus** - GitHub Discussions 集成
- **Vercel** - 部署平台
